    ########### Data Download and reduction ################################
###################################################################################################################################

- Carry out data download from CADC for SCUBA2 obs and Herschel HSA for PACS data.   

###### Input files required ####################

- .csv files containing URL list obtained from CADC for desired observations 
- Starlink version 2016B or later. In this paper we used Stralink version 2017A. 
- "myparams.ini" parameter file
- dimmconfig files

** See SCUBA2_Download+Reduction.txt and PACS_fitsFiles_Download.txt for more details **


#### .fits Files ####
1) .fits files of the sources listed in "Source_List_.." table for the chosen wavelength 


###### Output ####################

- Reduced SCUBA2 observations in .fits file form
